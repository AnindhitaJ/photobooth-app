LUX PHOTOBOOTH — EVENT TEMPLATE FILTER V4
========================================

TUJUAN
Pada halaman Pilih Template Event:
- Template yang tampil hanya template yang dipilih pada event.
- Kategori yang tampil hanya kategori dari template yang dipilih.
- Template tetap difilter sesuai tombol Photostrip atau Photobox 4R.
- Jika session/localStorage Safari hilang, template_ids event diambil ulang langsung dari Supabase.

FILE MINIMAL YANG WAJIB DITIMPA
1. template.html
2. sw.js

FILE LAIN DALAM PAKET
- event.html
- event-run.html
- result.html

Ketiga file tersebut adalah salinan kumulatif dari patch iPad Event V3. Timpa juga bila patch V3 belum pernah dipasang.

CARA PASANG
1. Timpa template.html dan sw.js di root repository.
2. Bila V3 belum dipasang, timpa seluruh file HTML dalam paket.
3. Commit dan deploy ulang ke Vercel.
4. Di iPad, tutup tab/PWA lama.
5. Hapus website data domain aplikasi atau hapus lalu pasang ulang PWA.
6. Buka Event, edit event, pastikan template yang dibutuhkan tercentang, simpan, lalu tekan Mulai.

HASIL YANG DIHARAPKAN
Contoh: event hanya memilih template kategori “baru1” dan “Photostrip”.
Halaman event hanya menampilkan tab:
- Semua
- baru1
dan hanya kartu template terpilih dalam kategori tersebut.
