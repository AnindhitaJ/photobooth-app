LUX PHOTOBOOTH — USER-OWNED TEMPLATE FILTER V5
=============================================

TUJUAN
Pada halaman /template biasa, hanya template yang memiliki:
  templates.user_id = ID user yang sedang login
yang ditampilkan, termasuk ketika user tersebut adalah super-admin.

PERUBAHAN
- Query Supabase ditambah filter user_id=eq.<JWT sub>.
- ID pemilik diambil dari JWT sesi, bukan sb_user_id yang mudah diubah.
- Hasil query difilter ulang di browser berdasarkan user_id.
- Template Built-in/Basic tidak lagi ditampilkan karena bukan milik user.
- Kategori hanya berasal dari template milik user.
- Kategori lama dari localStorage tidak ikut ditampilkan.
- Cache service worker dinaikkan agar template.html baru diterima perangkat.

FILE YANG WAJIB DITIMPA
1. template.html
2. sw.js

CARA PASANG
1. Timpa kedua file di root repository.
2. Commit dan deploy ulang ke Vercel.
3. Tutup semua tab/PWA lama.
4. Hapus website data domain aplikasi atau uninstall lalu install ulang PWA.
5. Login ulang dan buka /template.

CATATAN DATA
- Template lama yang user_id-nya NULL tidak akan tampil.
- Pastikan setiap template yang memang dimiliki user memiliki user_id yang benar.
- Patch ini membatasi tampilan aplikasi. RLS database tetap harus benar untuk keamanan server-side.
