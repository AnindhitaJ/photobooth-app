LUX PHOTOBOOTH — GLOBAL BUILT-IN TEMPLATE FIX V1.7
=================================================

TUJUAN
- Template Built-in dan Basic tampil untuk semua user.
- Template upload dari Supabase tetap hanya tampil kepada pemiliknya:
  templates.user_id = ID user yang sedang login.

PERUBAHAN
- Daftar non-event sekarang menggabungkan template global dan template milik user.
- Kategori Built-in kembali ditampilkan.
- Kategori teknis __basic__ tidak dibuat menjadi tab, tetapi Basic tersedia pada tab Semua.
- Photobox Built-in kembali tersedia pada alur Photobox.
- Event mode tidak diubah: tetap hanya menampilkan template database yang dipilih pada event.
- Cache service worker dinaikkan ke lux-photobooth-v1.7-global-builtins.

FILE YANG WAJIB DITIMPA
1. template.html
2. sw.js

CARA PASANG
1. Timpa kedua file di root repository.
2. Commit dan deploy ulang ke Vercel.
3. Tutup semua tab/PWA lama.
4. Buka ulang aplikasi.
5. Bila template lama masih tampil, hapus website data domain aplikasi atau uninstall/install ulang PWA.

CATATAN KEAMANAN
Template upload tetap difilter menggunakan user_id dari subject JWT dan difilter ulang di browser.
Template Built-in tidak berasal dari database sehingga tidak memiliki user_id dan memang bersifat global.
