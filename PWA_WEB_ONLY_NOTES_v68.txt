Lux Photobooth Web/PWA Latest v68

Pakai ZIP ini untuk deploy ke Vercel.
Tidak perlu Android Studio dan tidak perlu APK.

Kamera:
- memakai kamera browser seperti sebelumnya
- cocok untuk kamera laptop/tablet/HP/webcam/capture-card yang kebaca sebagai camera device oleh browser

Deploy:
1. Extract ZIP
2. Upload/deploy folder web ke Vercel
3. Hard refresh browser/PWA setelah deploy

Catatan:
Kalau sebelumnya sudah install PWA di tablet/HP, buka ulang atau clear cache kalau masih kebaca versi lama.
